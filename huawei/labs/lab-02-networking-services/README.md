# Lab 02 – Networking Services Practice (Huawei Cloud)

## Goal
Build a VPC network and validate secure connectivity + internet access + load balancing.

## Services Used
- VPC
- Security Groups
- EIP
- ELB
- ECS

## Tasks Completed
- Creating VPCs
- Buying ECSs
- Communication between ECSs
- Traffic control by security groups
- Access to the Internet with an EIP
- Using ELB to distribute incoming traffic

## Evidence
- `evidence/Networking-Services-Practice-Report.pdf`

## Key Learnings
- VPC fundamentals
- East–West vs North–South traffic
- Security groups for access control
- Internet access via EIP and traffic distribution via ELB
